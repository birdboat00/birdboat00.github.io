eden

Abstract

eden is a minimalist bytecode virtual machine. It is designed as a portable target for programming languages.

Trying it out

Getting eden

Eden can be found on GitHub.

Saving the builtin test pack

eden features a builtin test pack which can be saved to a code.eden file:


  eden --savetestpack

Running a pack

eden bytecode can be run using the virtual machine:


  eden --load <packfilename.eden>
